Assassin's Creed: Director's Cut - Freezing Fix

Dec 31, 2025
Game: Assassin's Creed: Director's Cut
Steam AppID: 15100
Platform: Proton/Steam
Difficulty: Intermediate

◉ Problem

Game freezes every 30-60 seconds for 3-5 seconds during gameplay. Audio continues but video locks up. The freezing is caused by the game attempting to contact defunct Ubisoft servers (gconnect.ubi.com) for statistics upload. The connection times out, causing periodic stuttering.

◉ Solution: Hex Edit Executable

The most effective fix is to hex edit the game executable, nullifying the server connection string. This prevents the game from attempting any connection, eliminating timeout delays.

  1. Install hex editor
    sudo apt install hexedit
  2. Navigate to game directory
    cd ~/.steam/steam/steamapps/common/"Assassins Creed"/
  3. Backup original executable
    cp AssassinsCreed_Dx9.exe AssassinsCreed_Dx9.exe.backup cp AssassinsCreed_Dx10.exe AssassinsCreed_Dx10.exe.backup
  4. Open executable in hex editor
    hexedit AssassinsCreed_Dx9.exe
  5. Search for server string

    Press Tab to switch to ASCII view, then press / to search. Type:

    gconnect.ubi.com
  6. Replace with null characters

    Replace each character of "gconnect.ubi.com" (17 chars) with a period (.) which represents null:

    gconnect.ubi.com → .................
  7. Save and exit

    Press Ctrl+X, confirm with Y

  8. Repeat for DX10 version if needed
    hexedit AssassinsCreed_Dx10.exe

    Follow steps 5-7 again

Important: Do NOT delete characters, only REPLACE them. The file must maintain the same size. Each character must be replaced with a period (.) to properly null the string.
Why this works: The game has "gconnect.ubi.com" hardcoded in the binary. By nullifying this string, the connection fails instantly instead of waiting for a TCP timeout (3-5 seconds). The hosts file method doesn't work because it only prevents DNS lookup but still waits for connection timeout.

◉ Alternative: Hosts File Method (Less Effective)

This method reduces but doesn't eliminate freezing. Only use if hex editing isn't possible.

# Edit Proton prefix hosts file: cd ~/.steam/steam/steamapps/compatdata/15100/pfx/drive_c/windows/system32/drivers/etc/ nano hosts # Add these lines: 127.0.0.1 gconnect.ubi.com 127.0.0.1 uts.ubi.com
